# AtCoder Weekday Contest 参加記録
[AtCoder](https://atcoder.jp/)で開催されている「AtCoder Weekday Contest」の解答コードを管理するリポジトリです。

## このリポジトリについて
このコンテストは 2026/02 現在Beta版として運用されており、公式ルールにて「コンテスト中の生配信、SNSへの投稿、AIの利用」などが許可されています。
本リポジトリでは試験中にコード公開をする可能性がありますが、現時点に置いて規約に違反しておりません。

> **公式ルール（引用）:**
> 「このコンテストはBeta版です。現段階では、AIの利用や、コンテスト中の生配信やXへのポストなどを許可します。Beta版で無くなり次第、ルールが変わる可能性が高いです。」

## 構成
- `week*`: 第 * 週の問題に対する解答コード
- `week*/memo/`: 解法のメモや計算量の考察
